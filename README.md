Badofone Support is a **satirical AI-powered chat system** that delivers the worst customer service experience possible. Get ready for sarcastic, unhelpful, and downright frustrating bot responses as you struggle to get your issues resolved!

You can change autoanswers, or you can fully launch this in the real mode of bad assistance.

## Features
- **Fully automated chatbot** that gives hilariously bad responses.
- **Randomized bot replies** to ensure you never get real help.
- **Terrible user experience** simulating corporate tech support.
- **Fake escalation system** that does nothing.
- **UI Components** using TailwindCSS and Material UI.

## Tech Stack
- **Next.js** (React framework)
- **TypeScript** (for type safety)
- **TailwindCSS** (for styling)
- **Material UI** (for UI components)
- **Docker** (for containerization)
